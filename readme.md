## 1、说明

该程序实现了自动记录鼠标和键盘的点击次数。

在页面上可以查看当天点击鼠标和键盘的次数，以及每个小时的点击数量信息。

开机自启动

不支持关闭程序：这个真是个傻逼功能。

使用logback这个日志工具，log4j2无法打包成功。

支持历史数据查询
